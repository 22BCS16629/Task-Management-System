Task Management System

A full-stack task management application developed using Java, React, Apache Maven, and Apache Tomcat. The application enables users to create, update, view, and delete tasks through an interactive web interface.

Features

* Create, update, and delete tasks
* Interactive browser-based interface
* Task management using CRUD operations
* Modular frontend and backend architecture
* Maven-based project management

Tech Stack

Frontend

* React.js
* HTML
* CSS
* JavaScript

Backend

* Java
* Apache Maven
* Apache Tomcat

Project Structure
Task-Management-System

├── backend
└── frontend

Learning Outcomes

This project demonstrates full-stack web development concepts, Java application development, client-server interaction, CRUD operations, and modular software design.
